Introducción

Las ecuaciones de segundo grado son un tema fundamental del álgebra, ya que permiten resolver problemas donde la incógnita aparece elevada al cuadrado.
Este tipo de ecuaciones se utilizan en diferentes áreas, como la física, la ingeniería y la geometría, para modelar situaciones reales.
A diferencia de las ecuaciones de primer grado, las de segundo grado pueden tener dos soluciones, una sola solución o ninguna solución real.

¿Qué es una Ecuacion ?

Una ecuación de segundo grado es una ecuación algebraica que tiene la forma general ax² + bx + c = 0, donde a, b y c son números reales
y el valor de a no puede ser cero. El término ax² indica que la incógnita está elevada al cuadrado, lo que le da el nombre a este tipo de ecuaciones.
Resolverlas implica encontrar los valores de x que cumplen con la igualdad

Definiciones y fundamentos matemáticos

  • incognita a la primera potencia:La variable no está elevada a ninguna potencia mayor
  • Igualdad:Tienen un signo de igual (=) que separa dos expresiones
  • Coeficiente:Los números que acompañan a la incógnita se llaman coeficientes.
  • Termino independinet: Es el número que no está acompañado por la incógnita.

Ejemplos resueltos

Ecuación: x² + 7x + 10 = 0

Ecuación: x² + 7x + 10 = 0

Paso 1: a = 1, b = 7, c = 10

Paso 2: Aplicar la fórmula general

Paso 3: b² − 4ac = 49 − 40 = 9

Paso 4: x = (−7 ± √9) / 2

Paso 5: x₁ = −2, x₂ = −5

Ejemplo 2

Ecuación: 2x² − 8x + 6 = 0

Paso 1: a = 2, b = −8, c = 6

Paso 2: b² − 4ac = 16

Paso 3: x = (8 ± √16) / 4

Paso 4: x₁ = 3, x₂ = 1

Ejemplo 3

Ecuación: x² − 6x + 9 = 0

Paso 1: a = 1, b = −6, c = 9

Paso 2: b² − 4ac = 0

Paso 3: x = 3 (solución doble)
